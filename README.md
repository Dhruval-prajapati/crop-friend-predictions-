# 🌾 AgroSmart Pro - Intelligent Agriculture Platform

## Overview
AgroSmart Pro is an AI-powered agriculture platform that helps farmers maximize crop yields using machine learning predictions, real-time weather data, and smart recommendations.

## Features
- 🤖 **ML Crop Yield Prediction** using Random Forest
- 🗺️ **Interactive India Map** with state-wise soil data
- 💬 **AI Agriculture Assistant** chatbot
- 📊 **Analytics Dashboard** with charts
- 📋 **Government Schemes** information
- 🛒 **Agri Shop** with Amazon affiliate links
- 📰 **Agriculture News** updates
- 🌐 **Multi-language** (English, Hindi, Gujarati)

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript, Leaflet.js, Chart.js
- **Backend**: Python, Flask, scikit-learn, pandas
- **ML Model**: Random Forest Regressor
